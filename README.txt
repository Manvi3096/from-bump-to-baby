From Bump to Baby — Option B Complete Doctor Version

Features:
1. LMP fixed to 15 October 2025.
2. Custom date range filter for kicks with full timestamps.
3. Export options:
   - Doctor PDF report
   - CSV kick export
   - Full JSON backup
   - JSON import/restore
4. Editable delivery/C-section date.
5. Health log for bile acids, SGOT, SGPT, fasting sugar, itching, CTG, notes and Udiliv.
6. Fixed Time-to-10 calculation for current session only.
7. PWA install support for iPhone.

GitHub Pages:
- Extract ZIP.
- Upload files to repo root.
- Settings > Pages > Deploy from branch > main > /(root).
- Open URL on iPhone Safari > Share > Add to Home Screen.
